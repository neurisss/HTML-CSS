Principais Formatações

Negrito / Destaque

Nessa frase temos termo em negrito usando a tag <b> (Não Semântica)

Nesta frase temos termo em destaque usando a tag <strong>

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (Não Semântica)

Nesta frase, temos um termo em Ênfase usando a tag <em>

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo , temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos inserir um texto como excluído para indicar que ele deve ser lido mas não considerado <del>

texto inserido

Podemos inserir um texto como inserido para indicar que ele foi adicionado depois <ins>

Existe também o sublinhado com a tag U (Não Semântica)

Texto sobrescrito

Para inserir coisa do tipo x20+3 com a tag <sup> ex: <sup>20<sup>

Texto subscrito

Para inserir coisa do tipo h2o com a tag <sub> ex: <sub>2<sub>
